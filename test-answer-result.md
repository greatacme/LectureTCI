# 테스트
테스트의 질문과 결과의 척도는 1:1아니며, 여러개의 질문이 하나의 척도와 매핑된다.
테스트는 한가지만 존재한다.
테스트는 여러개의 질문들로 이루어진다.
각 질문항목들은 척도를 속성으로 가진다
응답은 참여자별로 질문별로 저장된다
